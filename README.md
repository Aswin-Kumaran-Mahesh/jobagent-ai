# JobAgent AI — Web Interface

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open your browser at:
   http://localhost:5000

4. Enter your Anthropic API key in the UI and click Run.

## Files
- app.py          — Flask backend + agent logic
- jobs.csv        — 25 AI/ML job postings dataset
- templates/      — HTML frontend
- requirements.txt
